# Laundry Wallah / Laundry Mart - HTML Webpage

## Description
This project is a simple HTML webpage created for the Laundry Wallah / Laundry Mart service. The webpage provides information about the laundry services offered, pricing details, and a booking form for customers.

## Features
- Welcome header for the laundry service
- About section describing the service
- Image representing laundry service
- List of available services
- Price list displayed in a table
- Booking form with name, email, and phone fields
- Footer with contact information

## Technologies Used
- HTML

## How to Run
1. Download or clone the project folder.
2. Open the `index.html` file in any web browser.
3. The webpage will display the Laundry Wallah / Laundry Mart website.
